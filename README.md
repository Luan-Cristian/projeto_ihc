📚 BookAccess

Sistema web desenvolvido para a disciplina de Interação Humano-Computador (IHC), com foco em acessibilidade digital para leitura de documentos PDF.

O objetivo do projeto é permitir que usuários carreguem arquivos PDF e tenham uma experiência de leitura mais acessível através de recursos como:

Tradução em Libras utilizando VLibras
Navegação por teclado
Ajuste de contraste
Controle de tamanho da fonte
Leitura de documentos PDF diretamente no navegador
Recursos de acessibilidade voltados para inclusão digital

📦 Instalação
Pré-requisitos

Antes de iniciar, instale:

Node.js (versão 18 ou superior)
npm

Verifique se estão instalados:

node -v
npm -v
🔧 Clonando o projeto
git clone <URL_DO_REPOSITORIO>

Entre na pasta:

cd projeto_ihc
📥 Instalando dependências

Instale todas as dependências do projeto:

npm install

Instale também as bibliotecas utilizadas:

npm install react-router-dom
npm install pdfjs-dist
▶️ Executando o projeto

Inicie o servidor de desenvolvimento:

npm run dev

A aplicação estará disponível em:

http://localhost:5173
